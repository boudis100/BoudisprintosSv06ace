O projektu

BoudisprintOS je komunitní rozšíření firmwaru Klipper pro tiskárny Sovol SV06 Ace a SV07. Nabízí optimalizované rozhraní, automatizace a nové funkce pro zkušené uživatele.

Funkce

  • Optimalizované Fluidd rozhraní (modro-černé téma)
  • Input Shaping s podporou ADXL345
  • PID Autotune
  • Notifikace (např. e-mail, MQTT)
  • OTA aktualizace
  • Komunitní startovací obrazovka

About the project

BoudisprintOS is a community Klipper firmware enhancement for Sovol SV06 Ace and SV07 printers. It provides an optimized interface, automation tools, and advanced features for experienced users.

Features

  • Optimized Fluidd interface (blue-black theme)
  • Input Shaping support (ADXL345)
  • PID Autotune
  • Notifications (e.g., email, MQTT)
  • OTA updates
  • Community welcome screen
